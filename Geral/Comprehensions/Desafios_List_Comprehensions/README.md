````markdown
# Desafios de List Comprehensions

Nesta pasta estão todos os desafios realizados para treinar **List Comprehensions** em Python.

## Lista de Desafios

1. **Quadrados de números pares ou ímpares**  
   Elevar os números ao quadrado aplicando filtros.

2. **Letras de uma frase em maiúsculas**  
   Ignorar espaços e transformar cada letra.

3. **Multiplicação de valores da lista**  
   Alterar os valores multiplicando por um número.

4. **Tuplas (n, n²) ou (n, n³)**  
   Criar listas de tuplas com números e suas potências.

5. **Filtragem de palavras por tamanho**  
   Criar listas apenas com palavras maiores que um determinado número de caracteres.

6. **Exercícios combinados**  
   Mistura de filtros, transformações e tuplas.

## Como Executar

```bash
cd list_comprehensions
python nome_do_arquivo.py
```
````

Todos os desafios foram feitos como exercícios de fixação, com foco em criar listas eficientes e compactas.
